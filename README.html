Skytale

Bei diesem Verfahren schreibt man den Text spaltenweise statt zeilenweise auf und gibt ihn zeilenweise weiter. Der Schlüssel entspricht hier also der Anzahl der Zeilen.
Skytale ist das älteste (heute bekannte) Verschlüsselungsverfahren, bereits die Spartaner nutzen es. Sie wickelten wahlweise ein Leder- oder Pergamentband um einen Holzstab (Skytale von griechisch „Stab“), um die Nachricht zu verfassen.



Zum Entschlüsseln wird eine identische Skytale benötigt. Der Schlüssel war somit der Durchmesser des Stabs.
Das sehr unsichere Verfahren ist bis heute wichtig. Oft wendet man dieses auf einen bereits verschlüsselten Text an, um das Decodieren zu erschweren.
